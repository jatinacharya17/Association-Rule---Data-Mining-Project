# Association Rule Mining using Apriori Algorithm

A Data Mining project that implements the Apriori Algorithm to discover hidden relationships between student course enrollment patterns. The system reads transaction data from an Excel file (`data.xlsx`), converts it into a one-hot encoded format, generates frequent itemsets, and extracts association rules using Support, Confidence, and Lift metrics.

## Features

* Excel-based transaction dataset input (`data.xlsx`)
* Menu-driven Python application
* Automatic transaction processing
* One-Hot Encoding using TransactionEncoder
* Frequent Itemset Generation using Apriori Algorithm
* Association Rule Generation
* Support, Confidence, and Lift Calculation
* Interactive threshold selection
* Error handling and data validation
* Educational implementation of Data Mining concepts

## Technologies Used

* Python
* Pandas
* Mlxtend
* OpenPyXL
* Excel Dataset

## Project Structure

* `association_rule.py` – Main Python implementation
* `data.xlsx` – Transaction dataset
* `Project_Report.docx` – Complete project documentation

## Learning Outcomes

This project demonstrates practical implementation of Association Rule Mining, Frequent Pattern Discovery, Market Basket Analysis concepts, and Data Mining techniques using Python.
